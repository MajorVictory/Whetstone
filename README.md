## Installation

### Requirements

**[Core Compatibility]**:
 * *FoundryVTT* 0.6.5-0.6.6

**[Known Issues]**
 * *Unknown Compatibility* with game specific sheets and dialogs
 * [Report Problems](https://github.com/MajorVictory/Whetstone/issues)
